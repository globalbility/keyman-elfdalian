Elfdalian keyboard layout

The Elfdalian layout is based on the Swedish keyboard layout. All key sequence descriptions are based on the printed keys on a Swedish physical keyboard.

For KeymanWeb version of the layout, extended Latin characters are accessed via Ctrl+Alt key sequences. Keyman Desktop (Windows) and Keyman for Mac OSX use the AltGr and Option keys.
